				PYTHON

Python is a high-level, general-purpose programming language designed for simplicity and readability. Developed by Guido van Rossum
and first released in 1991, Python has since grown into one of the most popular programming languages due to its ease of use, versatility,
and large community.

Key Features:
	1. Easy to Learn and Use
	2. Interpreted Lanuage
	3. Dynamic Typing
	4. Large Standard Library
	5. Cross-Platform
	6. Object-Oriented and Functional

Use Cases:
	1. Web development: Django, Flask.
	2. Data Science and Machine Learning: Numpy, Pandas, Tensorflow and PyTorch.
	3. Game Developement: Pygame.
